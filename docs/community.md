# Community

GitHub Discussions are enabled for TailChase.

## Use Discussions For

- usage questions
- workflow ideas
- prompt-quality feedback
- agent export feedback
- release and installation questions
- broader design discussion before an issue is ready

## Use Issues For

- reproducible bugs
- focused documentation fixes
- collector parsing edge cases
- redaction misses
- small, scoped implementation tasks

Good-first issues are drafted in
[docs/good-first-issues.md](good-first-issues.md), but they have not been
opened yet. Issue numbers should not be referenced until maintainers create the
issues manually.

## Useful Contribution Areas

- docs
- fixtures
- agent exports
- redaction
- collector edge cases
- prompt quality
