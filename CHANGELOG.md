# Changelog

All notable changes to TailChase are documented here.

## [0.1.28] - 2026-06-26

- Split CI, release, nightly, and Scorecard workflows for clearer release gates.

- Added golden-style artifact and no-network core flow test coverage.

### Changed

- Repositioned the README around three core workflows and clearly labeled experimental surfaces.

## [0.1.27] and Earlier

- Early alpha development of GitHub Actions, GitLab CI, local evidence, report, export, guard, MCP, run-loop, and tournament commands.
